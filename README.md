# Simple C++ Project

This project demonstrates a simple C++ program where `main` calls a function with an integer parameter and returns the number as a string.

## Files
- `src/main.cpp`: Entry point of the program.
- `src/number_to_string.cpp` and `src/number_to_string.h`: Function implementation and declaration.
- `Makefile`: Build instructions.

## Build and Run

```bash
make all
build/main <int number> <group name>
build/main 100 MY_GROUP

make coverage
```
Coverage report generated in coverage_report/index.html
